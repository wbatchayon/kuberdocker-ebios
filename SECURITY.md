# Security Policy - Kuberdocker EBIOS

## Politique de Securite

Ce projet applique la methodologie EBIOS Risk Manager pour l'analyse des risques cybersecurite.

### Analyse EBIOS-RM Realisee

| Atelier | Statut | Mesures Identifiees |
|---------|---------|---------------------|
| Atelier 1 | Complet | 5 valeurs metier, 8 referentiels |

| Atelier 4 | Complet | 10 etapes MITRE ATT&CK |
| Atelier 5 | Complet | Reduction 62.5% du risque |

### Mesures de Securite Recommandees

- MFA obligatoire pour tous les acces admin
- EDR sur les postes critiques
- Segmentation reseau
- Sauvegardes immutables (regle 3-2-1)
- SIEM avec detection comportementale
- Formation sensibilisation trimestrielle
- Audit annuel des fournisseurs

### Signalement de Vulnerabilites

Si vous trouvez une vulnerabilite reelle dans ce code :

1. Ne pas creer d'issue publique
2. Contacter directement l'auteur
3. Decrire le probleme en detail
4. Attendre la correction avant toute divulgation

### Conformite

- RGPD - Protection des donnees personnelles
- ISO 27001 - Management de la securite
- Guide ANSSI - Bonnes pratiques de securite
